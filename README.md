# `cuda-device-query`

`cuda-device-query` is a Rust port of [`deviceQuery.cpp`](https://github.com/NVIDIA/cuda-samples/blob/master/Samples/1_Utilities/deviceQuery/deviceQuery.cpp) written in Rust with [`cudarc`](https://github.com/chelsea0x3b/cudarc).

```bash
cargo install cuda-device-query
cargo run cuda-device-query
```
